## ✨ The Experience

ChitChat is a **production-ready** communication platform that bridges the gap between clean aesthetics and high-performance engineering.

### 🌟 Key Highlights
*   **⚡ Instant Sync:** Powered by **Socket.io** for real-time message delivery.
*   **🔐 Industrial Security:** Secure JWT authentication via HTTP-only cookies.
*   **👀 Smart Feedback:** Integrated **Seen/Delivered** status & **Live Typing** indicators.
*   **🖼️ Media Optimization:** High-speed image sharing powered by **Cloudinary**.
*   **🧠 Reactive State:** Lightweight, hyper-fast state management with **Zustand**.
*   **🎨 Personalized UI:** A suite of dynamic themes for a tailored user experience.

## 🚀 Quick Start

### 1️⃣ Clone & Navigate
```bash
git clone https://github.com/yourusername/chitchat.git
cd chitchat
```

### 2️⃣ Initialize Environment
Create a `.env` file in the `/backend` directory:
```env
PORT=5001
MONGODB_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
CLOUDINARY_CLOUD_NAME=your_name
CLOUDINARY_API_KEY=your_key
CLOUDINARY_API_SECRET=your_secret
```

### 3️⃣ Launch the Application
```bash
# Terminal 1 (Backend)
cd backend && npm install && npm run dev

# Terminal 2 (Frontend)
cd frontend && npm install && npm run dev
```
